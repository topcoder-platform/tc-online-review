Question x.x.x The component specification includes a clear and easily grasped explanation of algorithms used in the design.

The reviewer should judge whether the algorithm descriptions are complete, accurate and contain enough information for the developer to properly implement the algorithms. Please note that this review question should judge only the correctness of the algorithm and how well the designer relates the algorithm [\u150] the grammar and spelling should NOT be included in this question. Question x.x.x The sequence diagrams cover complicated use cases or other complicated sequences where the interactions of the objects are not obvious.

Sequence diagrams should be included when there are interactions between objects that need to be demonstrated or where interactions are not immediately obvious or where method documentation may benefit from a supplemental diagram.

Rating 4 [\u150] All [\u145]complicated[\u146] interactions have been described by corresponding sequence diagrams, and none of the diagrams are incorrect or misleading. This rating permits minor issues such as mislabeled methods or mismatched parameters or return values to exist in the diagram.
